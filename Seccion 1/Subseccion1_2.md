# CSS

CSS es el lenguaje encargado de definir la apariencia visual del contenido estructurado con HTML.

Sus características principales son controlar colores, tamaños, tipografías, márgenes y más.

Funciona mediante selectores que apuntan a elementos HTML, aplicándoles reglas de estilo.

Los tipos de CSS son inline dentro de una etiqueta HTML, interno dentro de <"style">, externo (en un archivo .css, el más profesional)

Se usa para, crear el diseño gráfico del sitio, añadir estilos y animaciones sin JavaScript.
