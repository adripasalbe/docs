# HTML

HTML es el lenguaje fundamental de la World Wide Web. Su función principal es estructurar el contenido de una página: textos, imágenes, vídeos, enlaces, formularios y más.


Sus características principales basado en etiquetas, todo se escribe usando elementos como <"h1">, <"p">, <"img">, <"a">.


Es jerárquico los elementos pueden contener otros elementos, creando una estructura lógica.


En su evolución hacia HTML5 añadió nuevas capacidades como nuevas etiquetas semánticas: <"header">, <"footer">, <"article">, <"section">, audio y vídeo <"audio">, <"video">. Una mejor accesibilidad y una mejor integración con JavaScript.


Se usa en páginas web, aplicaciones web y documentación online.
